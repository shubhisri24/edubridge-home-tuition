# 🎓 EduBridge - Home Tuition Website

A modern, responsive home tuition website built using **React**, **Vite**, and **Tailwind CSS**.

This project was developed as a real-world website for a home tuition service, allowing parents and students to explore tutoring services and register for personalized home tuition.

---

## ✨ Features

- Responsive Landing Page
- Modern Hero Section
- About Section
- Why Choose Us
- Subjects We Teach
- Tutor Profile
- Contact Section
- Smooth Scroll Navigation
- Mobile Friendly Design

---

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- React Icons

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/shubhisri24/edubridge-home-tuition.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
